STEP-BY-STEP: Make the repo on your GitHub account and push the prepared files

1) Create the repo on GitHub (web UI)
   - Go to https://github.com/new
   - Repository name: ahmed-first-pro
   - Description: A simple GitHub workflow demo project (prepared for assignment)
   - Public (or Private if required)
   - DO NOT check "Initialize this repository with a README"
   - Click "Create repository"

2) On your machine, open terminal and run these commands inside the folder that contains the prepared files:

# Replace YOUR_GITHUB_USERNAME with your GitHub username in the command below.
git init
git add .
git commit -m "Initial commit with project files"
git branch -M main
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/ahmed-first-pro.git
git push -u origin main

-------------------------
BRANCH / FEATURE WORK
-------------------------
# Create a feature branch, make some changes, and push
git checkout -b feature/update-readme
# edit README.md locally (optional) — you can add a short line, then:
git add README.md
git commit -m "Add project description and usage instructions"
git push -u origin feature/update-readme

# Add cards:
